# Telegram HTML отчёт по адресу и компактный лист

Изменено:
- HTML-файл для Telegram называется по адресу замера;
- если адрес не указан, используется номер заказа;
- HTML-отчёт сделан компактным под один лист A4;
- схемы окон больше не выделяются отдельными карточками/блоками;
- данные и схемы расположены плотнее.
