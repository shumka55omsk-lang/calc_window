Закуп материалов

Для подбора ширины рулона к стороне, которая попадает в рулон, автоматически добавляется запас 9 см. Если ширина больше 2,4 м, но высота с запасом входит в рулон, калькулятор разворачивает раскладку. SVG для ЧПУ строится по оптимизированному раскрою внутри каждого рулона. Для каждого рулона можно получить отдельный SVG, а в деталях проставляются размеры и подписи.
